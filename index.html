Breve História da Computação.

A história da computação começou com instrumentos de cálculo como o ábaco e evoluiu com invenções mecânicas de Pascal e Babbage. No século XX, surgiram os primeiros computadores eletrônicos com válvulas, seguidos por transistores e circuitos integrados. A chegada dos microprocessadores permitiu o surgimento dos computadores pessoais. Atualmente, a computação é marcada por inteligência artificial, internet e avanços como computação em nuvem e quântica.

O que são Clients?

Na computação, um client (cliente) é qualquer dispositivo ou programa que envia solicitações para outro programa, geralmente chamado de servidor. O client consome os serviços fornecidos por esse servidor.

Exemplos comuns de clients incluem:

  • Navegadores web como Chrome, Firefox, Edge
  • Aplicativos móveis
  • Softwares de e-mail

O client é responsável por interagir com o usuário final, capturar ações e exibir os resultados.

O que são Servers?

Um server (servidor) é um sistema que fornece serviços, recursos ou dados para outros dispositivos chamados clients.

O servidor responde às solicitações dos clients, enviando os dados necessários ou executando ações.

Exemplos de servidores:

  • Servidor Web (como Apache ou Nginx)
  • Servidor de Arquivos
  • Servidor de Banco de Dados

Os servers podem estar na nuvem, em data centers ou até em redes locais.

Como é uma estrutura básica de um HTML?

Um documento HTML possui uma estrutura padrão que deve ser seguida. Veja abaixo um exemplo básico:

        <!DOCTYPE html>
        <html lang="pt-BR">
          <head>
            <meta charset="UTF-8">
            <title>Título da Página</title>
          </head>
          <body>
            <h1>Conteúdo Principal</h1>
            <p>Parágrafo de exemplo.</p>
          </body>
        </html>

Cada parte tem sua função:

  • <!DOCTYPE html>: informa ao navegador que este é um documento HTML5.
  • <html>: raiz do documento.
  • <head>: contém metadados, título, links de estilos, etc.
  • <body>: onde está o conteúdo visível da página.

Entendendo a diferença entre listas ordenadas e não ordenadas.

No HTML, há dois tipos principais de listas:

Lista Ordenada (<ol>)

Usada quando a ordem dos itens importa, como em instruções passo a passo:

  1. Pré-aqueça o forno
  2. Coloque o bolo
  3. Espere 40 minutos

Lista Não Ordenada (<ul>)

Usada quando a ordem não é relevante, como em uma lista de compras:

  • Leite
  • Pão
  • Açucar Queijo

Ambas usam o elemento <li> para os itens.
